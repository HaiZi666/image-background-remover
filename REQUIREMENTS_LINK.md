# Image Background Remover MVP 需求文档

> 创建日期：2026-03-29
> 状态：初稿

---

## 1. 项目概述

**项目名称**：Image Background Remover
**项目类型**：图片处理工具 Web 应用
**一句话描述**：用户上传图片，自动移除背景并支持下载透明背景 PNG
**目标用户**：设计师、电商卖家、内容创作者

---

## 2. 技术架构

### 2.1 技术栈

- **前端**：原生 HTML + JavaScript（无框架依赖）
- **样式**：TailwindCSS（CDN 引入）
- **后端**：Cloudflare Worker（处理 API 中转）
- **图片处理**：全程内存流，不落盘
- **静态部署**：Cloudflare Pages

### 2.2 第三方 API

- **背景移除 API**：Remove.bg
  - API Key：由管理员配置
  - 免费额度：50 张/月
  - 超出费用：$0.2/张

### 2.3 数据流程

```
用户上传图片 → Cloudflare Worker
→ Base64 转发给 Remove.bg API
→ 收到结果（透明背景 PNG）
→ 返回给前端展示/下载
```

---

## 3. 功能需求

### 3.1 核心功能（MVP）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 图片上传 | 支持拖拽或点击上传（支持 JPG/PNG/WebP） | P0 |
| 背景移除 | 调用 Remove.bg API 处理图片 | P0 |
| 结果预览 | 显示处理后的透明背景图片 | P0 |
| 图片下载 | 一键下载透明背景 PNG | P0 |
| 加载状态 | 处理中显示 loading 动画 | P0 |
| 错误提示 | API 失败时友好提示用户 | P0 |

### 3.2 边界情况处理

| 场景 | 处理方式 |
|------|----------|
| 图片过大（>10MB） | 前端拦截，提示图片大小限制 |
| 非图片文件 | 前端拦截，提示格式错误 |
| API 调用失败 | 显示错误信息，允许重试 |
| Remove.bg 额度用完 | 显示额度耗尽提示 |

## 5. API 设计

### 5.1 Cloudflare Worker 接口

- **端点**：`/api/remove-bg`
- **方法**：POST
- **请求体**：

```json
Content-Type: application/json
{
  "image": "data:image/jpeg;base64,xxxxx"
}
```

- **响应**：

成功 (200):
```json
{
  "success": true,
  "result": "data:image/png;base64,xxxxx"
}
```

失败 (400/500):
```json
{
  "success": false,
  "error": "错误描述"
}
```

### 5.2 Worker 核心逻辑

```javascript
// 接收图片 → 转发给 Remove.bg → 返回结果
// 不存储任何数据，全程内存处理
```
